# 米生新闻 (NewsHub)

一键收集 7 个中文新闻源，本地 SQLite 存储，支持定时自动刷新和源筛选。

## 下载

[下载米生新闻 v1.0.0](https://github.com/Leason666/misheng-news/releases/download/v1.0.0/NewsHub.exe)

## 功能

- 一键收集：7 个新闻源同时抓取
- 指定范围：弹窗勾选要抓取的源
- 自动刷新：5/10/30/60 分钟可选
- 源筛选：按新闻源标签筛选
- 本地存储：SQLite 数据库持久化

## 内置新闻源

| 源 | 类型 |
|---|------|
| 开源中国 | 技术资讯 |
| 爱范儿 | 科技媒体 |
| 少数派 | 数字生活 |
| 阮一峰周刊 | 技术博客 |
| Hacker News | 英文技术 |
| Solidot | 科技新闻 |
| 云风 BLOG | 游戏开发 |

## 系统要求

- Windows 10/11 64位
- 无需安装 Python 或其他依赖

## 使用方式

双击 NewsHub.exe 即可启动。首次启动会自动创建数据库。
